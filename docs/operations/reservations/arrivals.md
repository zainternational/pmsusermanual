# Arrivals

The **Arrivals** view shows all reservations with an arrival date on the selected date. Use it to prepare for check-ins, confirm room readiness, and process arrivals efficiently.

## Access

From the sidebar: **Reservation** → **Arrivals** tab (or filter the reservation list by arrival date).

## What You See

Reservations with **arrival date** matching the selected date, showing:

- Reservation ID and guest name
- Room type and room number (if assigned)
- Arrival and departure dates, nights
- Reservation status (e.g. Reserved, Confirmed)
- Rate amount
- ETA (estimated arrival time) if set
- Special requests and alerts

## Preparing for Arrivals

Before checking in a guest:

1. Confirm the **room is assigned** — if no room number is set, assign one from the reservation or Room Plan.
2. Check **Housekeeping** to ensure the room is **Vacant Clean** (VC). A room that is Vacant Dirty cannot be checked in until cleaned.
3. Review any **alerts** or **special requests** on the reservation.
4. Settle any **pre-arrival deposit** if required by the reservation type.

## Checking In

From the arrivals list or the reservation detail:

1. Open the reservation.
2. Confirm all required fields are filled (room, rate, guest details).
3. Click **Check In**. The status changes to **In-House**.
4. The guest appears in **Front Desk → In-House**.

If the room is not ready, the check-in action may be blocked. Update the room status in Housekeeping first.

## Walk-In Check-In

For guests without a prior reservation, use **Housekeeping → Room Chart** to select a Vacant Clean room and click **Walk-In**. This opens the new reservation form pre-filled with the Walk-In reservation type, room, and today's arrival date.
