# Departures

The **Departures** view shows all in-house reservations with a departure date on the selected date. Use it to prepare folios, process check-outs, and hand rooms back to housekeeping.

## Access

From the sidebar: **Reservation** → **Departures** tab (or filter the reservation list by departure date).

## What You See

Reservations with **departure date** matching the selected date, showing:

- Reservation ID and guest name
- Room number and room type
- Arrival and departure dates, nights
- Reservation status (In-House)
- ETD (estimated departure time) if set
- Folio balance or link to billing

## Preparing for Departures

Before checking out a guest:

1. Open the reservation and review the **folio** in Cashiering → Transaction.
2. Post any outstanding charges (e.g. last night's room charge, minibar, late checkout fee).
3. Confirm the **folio balance** is zero or collect the remaining payment.
4. If the guest is paying by company or travel agent routing, confirm the routing is set up correctly.

## Checking Out

From the departures list or the reservation detail:

1. Open the reservation.
2. Ensure the folio balance is settled (zero or routed to a payer).
3. Click **Check Out**. The status changes to **Checked Out**.
4. The reservation moves to **Front Desk → Checked-Out**.
5. The room status updates to **Vacant Dirty** — housekeeping will clean it.

## Quick Checkout

For multiple departures, use **Cashiering → Quick Checkout** to process several reservations at once without opening each folio individually.

## Late Checkout

If a guest stays past the standard checkout time, post a late checkout charge from **Cashiering → Transaction** before checking out.
