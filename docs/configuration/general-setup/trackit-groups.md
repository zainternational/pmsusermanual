# Track It — Groups

**Track It Groups** are the top-level categories for items in the Lost & Found system. Every item registered in Lost & Found must belong to a group.

## Access

**Configuration → General Configuration → Track It → Groups**

## Fields

| Field | Required | Description |
|-------|----------|-------------|
| Short Code | Yes | Abbreviated code (e.g. ELEC, CLOTH, DOC). |
| Name | Yes | Category name (e.g. Electronics, Clothing, Documents). |
| Sequence | No | Display order in dropdowns. |

## Common Groups

| Short Code | Name |
|-----------|------|
| ELEC | Electronics |
| CLOTH | Clothing |
| DOC | Documents |
| ACC | Accessories |
| JEW | Jewellery |
| FB | Food & Beverage |
| OTHER | Other |

## Adding a Group

1. Click **Add**.
2. Enter a short code and name.
3. Set sequence if needed.
4. Save.
