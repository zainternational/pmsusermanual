# General Setup

General setup covers hotel information, system date, users, user groups, departments, job titles, license, cash register setup, TrackIt, and transportation types.

## Sub-items

| Sub-menu | Purpose |
|----------|---------|
| [Hotel Information](/docs/configuration/general-setup/hotel-information) | Property details, address, contact, time zone, logo. |
| [System Date](/docs/configuration/general-setup/system-date) | Business date used for reservations, rates, and reporting. |
| [Users](/docs/configuration/general-setup/users) | User accounts, permissions, property assignment. |
| [User Groups](/docs/configuration/general-setup/user-groups) | Permission sets and group assignment. |
| [Departments](/docs/configuration/general-setup/departments) | Departments for reporting or access control. |
| [Job Titles](/docs/configuration/general-setup/job-titles) | Job titles for users or profiles. |
| [License](/docs/configuration/general-setup/license) | License status and activation. |
| [Cash Register Setup](/docs/configuration/general-setup/cash-register-setup) | Cash registers for cashier login. |
| [TrackIt](/docs/configuration/general-setup/trackit) | Overview of the 4 Track It tables (Groups, Types, Actions, Locations) for Lost & Found. |
| [TrackIt — Groups](/docs/configuration/general-setup/trackit-groups) | Top-level item categories (Electronics, Clothing, Documents, etc.). |
| [TrackIt — Types](/docs/configuration/general-setup/trackit-types) | Sub-categories within a group (Phone, Laptop, Passport, etc.). |
| [TrackIt — Actions](/docs/configuration/general-setup/trackit-actions) | Actions taken on items (Found, Returned, Disposed, etc.). |
| [TrackIt — Locations](/docs/configuration/general-setup/trackit-locations) | Where items were found or stored (Lobby, Room 101, HK Storage, etc.). |
| [Transportation Types](/docs/configuration/general-setup/transportation-types) | Transportation types for pickup or guest transport. |
| [Services](/docs/configuration/general-setup/services) | Additional services offered to guests (transfers, laundry, spa, etc.). |

Access via **Configuration → General Configuration** (or equivalent sub-menus).
