# TrackIt (Groups, Locations, Actions, Types)

If **TrackIt** is used for maintenance or lost & found:

- **TrackIt groups** — Grouping of items or actions
- **TrackIt locations** — Where items are found or stored
- **TrackIt actions** — Types of actions (e.g. repair, replace)
- **TrackIt types** — Classification of items or tasks

Access via **Configuration → General Configuration → TrackIt** (or equivalent sub-menus).
