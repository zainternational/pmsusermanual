# Track It — Types

**Track It Types** are sub-categories within a Track It Group. They describe the specific kind of item (e.g. Phone within Electronics).

## Access

**Configuration → General Configuration → Track It → Types**

## Fields

| Field | Required | Description |
|-------|----------|-------------|
| Short Code | Yes | Abbreviated code (e.g. PHONE, LAPTOP). |
| Name | Yes | Type name (e.g. Phone, Laptop, Wallet, Passport). |
| Sequence | No | Display order in dropdowns. |

## Common Types by Group

| Group | Types |
|-------|-------|
| Electronics | Phone, Laptop, Tablet, Charger, Camera, Earphones |
| Clothing | Shirt, Jacket, Shoes, Hat, Scarf |
| Documents | Passport, ID Card, Visa, Boarding Pass, Credit Card |
| Accessories | Watch, Bag, Umbrella, Glasses, Keys |
| Jewellery | Ring, Necklace, Bracelet, Earrings |

## Adding a Type

1. Click **Add**.
2. Enter a short code and name.
3. Set sequence if needed.
4. Save.
