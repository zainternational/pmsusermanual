# Transaction Codes

- **Add / edit** transaction codes
- **Code**, **name**, **group** (or sub-group)
- **Type** (e.g. charge, payment, tax)
- **Posting rules** (e.g. room, inclusive, routing)
- **Mobile payment** or **card** flags if used

Transaction codes are used when posting charges on a folio.

Access via **Configuration → Cashiering → Transaction Code** (or equivalent).
