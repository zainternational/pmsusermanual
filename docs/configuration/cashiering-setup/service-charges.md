# Service Charges

- **Add / edit** service charges
- **Name**, **percentage** or **amount**
- **Transaction code** or **posting rule** (which charge it applies to)
- **Calculate on base amount** or **inclusive** if used

Service charges are auto-added to eligible charges (e.g. room, F&B).

Access via **Configuration → Cashiering → Service Charges** (or equivalent).
