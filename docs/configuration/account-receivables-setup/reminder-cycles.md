# Reminder Cycles

**Reminder Cycles** define how often payment reminders are sent to AR accounts with outstanding balances.

## Access

**Configuration → Account Receivables → Reminder Cycle**

## Fields

| Field | Description |
|-------|-------------|
| Name | Name of the reminder cycle (e.g. Weekly, Monthly, 30-Day). |
| Days | Number of days between reminders. |
| Active | Whether this cycle is available for selection. |

## Usage

Reminder cycles are assigned to AR accounts (company or travel agent profiles). When an account has an outstanding balance, the system uses the assigned cycle to determine when to send the next reminder.
