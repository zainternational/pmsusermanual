# Account Types

**Account Types** categorise AR (accounts receivable) accounts. Examples: Corporate, Travel Agent, Government, Group.

## Access

**Configuration → Account Receivables → Account Type**

## Fields

| Field | Description |
|-------|-------------|
| Name | The account type name (e.g. Corporate, Travel Agent). |
| Description | Optional description. |
| Active | Whether this type is available for selection. |

## Usage

Account types are assigned to company profiles and travel agent profiles when setting up their AR account. They are used for filtering and reporting in accounts receivable.
