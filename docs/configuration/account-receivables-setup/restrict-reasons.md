# Restrict Reasons

**Restrict Reasons** are the reasons used when placing a restriction or block on an AR account — for example, when a company has exceeded its credit limit or has an overdue balance.

## Access

**Configuration → Account Receivables → Restrict Reason**

## Fields

| Field | Description |
|-------|-------------|
| Name | The reason for the restriction (e.g. Overdue Balance, Credit Limit Exceeded, Account Suspended). |
| Description | Optional additional detail. |
| Active | Whether this reason is available for selection. |

## Usage

When restricting an AR account, the user selects a restrict reason from this list. The reason is recorded on the account and may appear in AR reports or alerts.
