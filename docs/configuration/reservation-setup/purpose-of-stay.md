# Purpose of Stay

**Purpose of Stay** codes classify why a guest is visiting — for example, Business, Leisure, Conference, or Transit. They are used on reservations for segmentation and reporting.

## Access

**Configuration → Reservation Setup → Purpose of Stay**

## Fields

| Field | Description |
|-------|-------------|
| Name | The purpose of stay label (e.g. Business, Leisure, Conference). |
| Short Code | Abbreviated code used in reports and filters. |
| Active | Whether this code is available when creating reservations. |

## Usage

When creating or editing a reservation, the agent can select a Purpose of Stay. This value is used in reservation reports and guest information reports for segmentation analysis.
