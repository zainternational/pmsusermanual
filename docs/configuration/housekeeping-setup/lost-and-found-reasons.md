# Lost & Found Reasons

- **Add / edit** lost & found reasons (e.g. Left in room, Lobby, Restaurant)
- **Name**, **code** (if used)

Used when registering found items in the Housekeeping Lost & Found tab.

Access via **Configuration → HouseKeeping & Maintenance** → **Lost & Found Reason** (or equivalent).
