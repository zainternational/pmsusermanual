# Housekeeping Setup

Housekeeping setup includes housekeeping sections, attendances, tasks, room management, lost & found reasons, management types, and out-of-order reasons.

## Sub-items

| Sub-menu | Purpose |
|----------|---------|
| [Housekeeping Sections](/docs/configuration/housekeeping-setup/sections) | Sections (e.g. Floor 1, Tower A) to group rooms and assign tasks. |
| [Housekeeping Attendances](/docs/configuration/housekeeping-setup/attendances) | Attendants or teams for task assignment. |
| [Housekeeping Tasks](/docs/configuration/housekeeping-setup/tasks) | Task types (e.g. Clean, Deep Clean, VIP Setup). |
| [Room Management](/docs/configuration/housekeeping-setup/room-management) | Room-to-section or zone assignment. |
| [Lost & Found Reasons](/docs/configuration/housekeeping-setup/lost-and-found-reasons) | Reasons for registering found items. |
| [Management Types](/docs/configuration/housekeeping-setup/management-types) | Room status or maintenance classification. |
| [Out-of-Order Reasons](/docs/configuration/housekeeping-setup/out-of-order-reasons) | Reasons for marking a room out of order. |
