# Room Management

- **Add / edit** room management records
- **Room** — Link to room
- **Section** or **zone** — Link to housekeeping section
- **Management type** or **classification** if used

Used to assign rooms to sections or logic for housekeeping.

Access via **Configuration → HouseKeeping & Maintenance** → **Room Maintenance Reason** (or equivalent).
