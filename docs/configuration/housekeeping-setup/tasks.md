# Housekeeping Tasks (Setup)

- **Add / edit** housekeeping task types (e.g. Clean, Deep Clean, VIP Setup)
- **Name**, **code** (if used)
- **Category** or **group** if used

Used when creating or assigning housekeeping tasks in the Housekeeping module.

Access via **Configuration → HouseKeeping & Maintenance** → **HK Task** (or equivalent).
