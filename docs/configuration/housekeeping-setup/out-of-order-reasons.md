# Out-of-Order Reasons

- **Add / edit** out-of-order reasons (e.g. Repair, Renovation)
- **Name**, **code** (if used)

Used when marking a room out of order in housekeeping or front desk.

Access via **Configuration → HouseKeeping & Maintenance** → **Out of Order/Service Reason** (or equivalent).
