# Rate Setup

Rate setup covers rates, rate setup, rate classes, rate categories, markets, sources, packages, currencies, and exchange rates.

## Sub-items

| Sub-menu | Purpose |
|----------|---------|
| [Rate Setup](/docs/configuration/rate-setup/rate-plan) | Rate plan (name, validity, conditions). |
| [Rate Classes](/docs/configuration/rate-setup/rate-classes) | Broad grouping of rates (e.g. BAR, Corporate). |
| [Rate Categories](/docs/configuration/rate-setup/rate-categories) | Finer grouping (e.g. Weekday, Weekend). |
| [Rates](/docs/configuration/rate-setup/rates) | Add or edit rates, room type, date range, amount. |
| [Markets](/docs/configuration/rate-setup/markets) | Market segment or channel (e.g. Direct, OTA). |
| [Sources](/docs/configuration/rate-setup/sources) | Booking source and source group. |
| [Packages](/docs/configuration/rate-setup/packages) | Packages with inclusions, validity, price or rate link. |
| [Currencies](/docs/configuration/rate-setup/currencies) | Supported currencies (code, symbol, decimals). |
| [Exchange Rates](/docs/configuration/rate-setup/exchange-rates) | Rate to base currency, validity. |
