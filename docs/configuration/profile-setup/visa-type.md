# Visa Types

**Visa Types** are the categories of visas that guests may hold — for example, Tourist, Business, Transit, Diplomatic. They are stored on individual guest profiles for immigration and reporting purposes.

## Access

**Configuration → Profile Setup → Visa Type**

## Fields

| Field | Description |
|-------|-------------|
| Name | The visa type name (e.g. Tourist Visa, Business Visa, Transit Visa). |
| Short Code | Abbreviated code. |
| Active | Whether this type is available for selection on guest profiles. |

## Usage

Visa type is recorded on an individual guest profile along with visa number, issue date, and expiry date. This data appears in the [Guest Information Report](/docs/operations/reports/guest-information) for immigration compliance.
