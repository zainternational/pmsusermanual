# Manual Updates

This page lists major updates to the Hotelium PMS user manual.

When we publish updates, we add a new row below with the version, date, and a short summary of what changed. Use it to see what's new or to match the manual to your product version.

## What's New

| Version | Date | Changes |
|---------|------|---------|
| **v1.2** | April 2026 | Major gap-fill update. New pages: Billing (Guest Folio) — full folio management with split, transfer, adjust, void, print, routing, checkout; Cash Register Login — step-by-step guide; Post Room Charge — manual charge posting workflow; Housekeeping Management — main tabbed screen overview; Housekeeping Room Plan — calendar grid; Reservation Confirmation Letter; Reservation Registration Form; Profile Workflows — Linking a Profile to a Reservation, Guest Checkout Workflow; Database Manager — multi-tenant DB management for admins; Services configuration; First-Time Property Setup — full onboarding checklist with all 7 setup steps and go-live verification. TrackIt split into 4 dedicated pages (Groups, Types, Actions, Locations). User Groups expanded with full permission inheritance model and per-module permission tables. Dashboard expanded with detailed card descriptions (Overview Stats, Available Rooms, Housekeeping, Revenue Chart, VIP Levels, Complimentary, Reservation Details). Operations Overview updated with all new modules. Cashiering Overview updated with Cash Register Login and Post Room Charge links. Reservations Overview updated with Confirmation Letter and Registration Form links. Housekeeping Overview updated with Management and Room Plan links. Empty scaffolding folders removed (tutorial-basics, tutorial-extras, profiles). |
| **v1.1** | April 2026 | Major documentation expansion. All operations modules fully documented with actual fields, filters, and columns from the application. New pages: Guest Information Report, Trial Balance Report, Manager Report, Journal by Transaction, Room Routing, Room Chart. Profile docs rewritten with all form fields (individual: 3-step form with travel documents; company and travel agent: address cascade, AR number, IATA). Reservation docs rewritten with full 3-step form (guest info, booking info, transportation), group reservations, daily rate override, activity panel. Front Desk docs rewritten with Room Plan calendar grid, In-House actions (room move, checkout flow, alerts), Checked-Out (move back to in-house). Cashiering docs rewritten with Transaction split-panel layout, all action buttons, checkout auto-trigger, Cashier Shift dashboard/close/reports, Quick Checkout batch processing. Housekeeping docs rewritten with all tab fields (Room Status bulk update, Tasks floor/room multi-select, Lost & Found Track It system, Maintenance OOO link, Out of Order return flow). Night Audit rewritten with pre-run confirmation modals (arrivals, departures, shift close), summary modal tabs, force logout wait. All 8 report categories fully documented with actual filters and columns. Configuration: Account Receivables Setup section added (4 pages), Visa Type, Purpose of Stay, Market Groups, Source Groups added. Glossary expanded with 20+ new terms. Troubleshooting expanded with module-specific entries. |
| **v1.0** | January 2025 | Initial release. Introduction, dashboard, operations (workflows, reservations, front desk, cashiering, housekeeping, night audit, reports, profiles), configuration, troubleshooting, glossary, and general topics. |
