# Manual Updates

This page lists major updates to the Hotelium PMS user manual. The footer shows the current manual version (**User Manual v1.0**).

When we publish updates, we add a new row below with the version, date, and a short summary of what changed. Use it to see what’s new or to match the manual to your product version.

## What’s New

| Version | Date       | Changes |

| **v1.0** | January 2025 | Initial release. Introduction, dashboard, operations (workflows, reservations, front desk, cashiering, housekeeping, night audit, reports, profiles), configuration, troubleshooting, glossary, and general topics. |
